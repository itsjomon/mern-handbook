# JS Object Literals
Used to store keyed collections & complex entities.

`property => (key, value)pair`

Objects are collection of properties

```js
const delhi = {
    latitude: "28.7041* N",
    longitude: "77.1025* E"
};
```

```js
let student = {
    name: "ria",
    age: 20,
    marks: 94.4,
    city: "Delhi"
};
```
